# Python for Astronomy

This repository is a personal collection of simple, hands-on Python tutorials focused on astronomy and scientific data analysis. I created this space to rebuild my core coding skills and reduce my dependence on AI by practicing each concept directly. The goal is to understand the fundamentals deeply and create a reference I can return to whenever I need it.

The tutorials here cover essential tools used in astronomy: reading and writing FITS files, plotting data with Matplotlib, working with NumPy arrays, exploring tables with Pandas, and handling common astronomy workflows such as spectra, cubes, and coordinate systems. Each example is intentionally small, clear, and practical, written from the perspective of someone learning step by step.

This repository is not meant to be a polished course. Instead, it is a living notebook of techniques, experiments, and reminders—things I want to understand independently and be able to reproduce without relying on automated tools. Over time, I will expand it with more examples, datasets, and techniques as I continue learning.

If you are also beginning your journey in Python for astronomy or trying to build stronger independent coding habits, you may find these examples useful. Feel free to explore, adapt, and build on anything here.
